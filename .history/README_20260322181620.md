# Matrix Service - Go Backend Challenge

A Go backend service that exposes REST endpoints, built for the Matrix backend challenge.  
This project includes CI/CD integration with **GitHub + CircleCI** and automated deployment to **AWS EKS**.

## Project Overview

Matrix Service is a lightweight Go service with the following features:

- Accepts file uploads via `/echo` endpoint  
- Runs on port `8080` inside the container  
- Designed for deployment to Kubernetes with Helm  
- Integrated with CircleCI for automated testing and deployment  

---

## Prerequisites

Before running locally or deploying:

- [Go 1.20+](https://golang.org/dl/)  
- [Docker](https://www.docker.com/)  
- [kubectl](https://kubernetes.io/docs/tasks/tools/)  
- [Helm 3+](https://helm.sh/docs/intro/install/)  
- AWS CLI configured with access to your EKS cluster  
- CircleCI account connected to GitHub repository  
- EKS cluster with public subnets tagged:

```text
kubernetes.io/role/elb = 1
kubernetes.io/cluster/<cluster-name> = shared
