## 📌 Overview

This project is a **production-ready Go microservice** that processes matrix data from CSV uploads and exposes REST endpoints for matrix operations.

While fulfilling the backend challenge requirements, this solution has been **extended to demonstrate real-world engineering practices**, including:

- CI/CD automation using CircleCI  
- Containerization with Docker  
- Kubernetes orchestration via AWS EKS  
- Helm-based deployment strategy  

---

## 🧠 Supported Operations

Given a square matrix CSV:

``` id="pi1d36"
1,2,3
4,5,6
7,8,9
```

| Operation   | Endpoint     | Output |
|------------|-------------|--------|
| Echo       | `/echo`     | Original matrix |
| Invert     | `/invert`   | Transposed matrix |
| Flatten    | `/flatten`  | Single row |
| Sum        | `/sum`      | 45 |
| Multiply   | `/multiply` | 362880 |
